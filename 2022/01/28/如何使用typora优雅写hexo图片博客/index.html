1. 部署hexo

1.1. git和node.js安装

git安装，直接官网下载安装git-osx-installer。如果安装了Homebrew，也可以参考如下操作。

node.js安装，以macOS为例，直接官网下载，

1.2. 生成博客

Hexo 是一个快速、简洁且高效的博客框架。接下来只需要使用 npm 即可完成 Hexo 的安装

1.3. 初始化本地文件夹

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

表示项目名称，可以自定义，然后 cd 表示进入到生成的项目，npm install 表示安装依赖。

2. 与GitHub建立关联

在GitHub上新建仓库 xxx.github.io，需要注意的是xxx必须是GitHub的账户名

然后项目就建成了，点击Settings，向下拉到最后有个GitHub Pages，点击Choose a theme选择一个主题。然后等一会儿，输入网址 xxx.github.io

4. 与编辑器联动

总体还是倾向于vscode这种all in的编辑器

4.1. 与typora进行联动

需要明确的是，hexo中博文存放在source/_posts文件夹中，而要在网页上显示markdown中的图片，需要以相对的位置。
这里将图片存放在source/img文件夹下，并且给个博文的图片单独存放在以博文命名的文件夹中
